Aula do Dia: 13/10/2023 Desenvolvimento web Regras comuns de CSS
`display flex` Imagine que seu site é como uma bandeja de comida. Com isso, você organiza os pratos de um jeito que fique bacana, mesmo se a bandeja for pequena ou grande. `color` A propriedade color define a cor do texto dentro de um elemento. Pode receber valores como nomes de cores, códigos hexadecimais ou RGB, permitindo personalizar a aparência do texto. `background-color` Agora, pense na cor de fundo do papel onde você está escrevendo. `background-color` faz isso para o seu site. Deixe do jeito que você acha legal. As regras CSS color e background-color aceitam uma variedade de padrões de cor para definir a cor de texto e o fundo de um elemento HTML, respectivamente. Nome da Cor: Palavras-chave que representam cores, como "red", "blue", "green", entre outros. Alguns exemplos incluem: red, blue, green, black, white, yellow, etc. Valores Hexadecimais: Valores hexadecimais de seis dígitos que representam uma cor. Por exemplo, #FF0000 representa vermelho, #00FF00 representa verde e #0000FF representa azul. Valores RGB: Pares de valores numéricos que representam a intensidade das cores vermelha (R), verde (G) e azul (B). Podem ser definidos usando rgb() ou rgba() (com um valor adicional para alfa/transparência). Exemplo de rgb(): rgb(255, 0, 0) para vermelho puro. Exemplo de rgba(): rgba(255, 0, 0, 0.5) para vermelho com 50% de transparência. `text-align` Define o alinhamento horizontal do texto dentro de um elemento. Pode ser left (esquerda), right (direita), center (centro) ou justify (justificado). `align-items` Com align-items, você coloca as coisas no meio da tela, como ajustar o posicionamento vertical das coisas no seu site. Alguns valores comuns para `align-items` incluem: `flex-start`: Alinha os itens no início do container transversalmente `flex-end`: Alinha os itens no final do container transversalmente `center`: Centraliza os itens transversalmente dentro do container `stretch`: Estica os itens para preencher todo o espaço transversalmente `baseline`: Alinha os itens de acordo com a linha de base do texto `justify-content` A propriedade CSS justify-content é usada para alinhar itens (como flex items) ao longo do eixo principal de um container flexível. Alguns valores comuns para justify-content incluem: `flex-start`: Alinha os itens ao início do container `flex-end`: Alinha os itens ao final do container `center`: Centraliza os itens dentro do container `space-between`: Distribui os itens igualmente com espaços entre eles, mas não antes do primeiro ou depois do último item `space-around`: Distribui os itens com espaços iguais em volta deles, inclusive antes do primeiro e depois do último item `space-evenly`: Distribui os itens com espaços iguais entre eles, incluindo antes do primeiro e depois do último item
